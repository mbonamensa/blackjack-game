Game Rules

The goal of the game is to get a BlackJack, which means having cards that sum up to 21. If you go above 21, you are out of the game and you can start again. If you're below 21 you can draw a new card. If you get a BlackJack(a sum of 21), you get $100! On screen though :) Click START GAME to begin
